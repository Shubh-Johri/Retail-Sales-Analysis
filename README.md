# Retail Store Sales Analysis

## Overview
This project analyzes retail sales data of electronic products across multiple cities (Pune, Mumbai, Nagpur) to extract meaningful business insights. The analysis is performed using Excel, Python, and SQL.

---

## Problem Statement
A retail company is facing fluctuations in sales and wants to identify:

- Top-performing products  
- Highest revenue-generating cities  
- Category-wise performance  
- Sales trends over time  

---

## Tech Stack
- Microsoft Excel (Pivot Tables)  
- Python (Pandas, Matplotlib)  
- SQL

## Key Features
- Calculation of total revenue  
- Product-wise sales analysis  
- City-wise revenue comparison  
- Category distribution analysis  
- Data visualization using charts  

---

## Results and Insights
- Best Product: Laptop (₹150000 revenue)  
- Top Cities: Mumbai and Pune (₹110000 each)  
- Accessories generate lower revenue compared to electronics  
- Sales are concentrated in major urban areas
